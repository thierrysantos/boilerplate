### Installing

A step by step series of examples that tell you how to get a development env running

Cloning project

```bash
git clone https://github.com/thierrysantos/gobarber-api.git
```

Installing project depencies

```bash
yarn
# or
npm install
```

Start project

```bash
yarn dev
# or
npm run dev
```
